# spring-api-aws

A professional API delivery in the cloud

API para controle de entreda e saída 
de veículos para um estacionamento.

* Banco de dados relacional
* Nuvem para publicação
* Controle de acesso
* Publicação Heroku

* Spring Boot 3
* Java 17
* Heroku
* Spring Cloud
* Spring Security
